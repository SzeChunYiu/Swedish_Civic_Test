## Distributed safety minimum (single host — mac-mini)

One canonical checkout. No worktree split needed for Batch 0.
All workers use main checkout with branch `batch/2026-05-15-foundation`.

## Iteration rule

Each lane: re-read this file + lane spec → check pwd/branch → confirm lease →
pick one task from queue → confirm task maps to TEAM_PLAN checklist item →
make focused changes in writable scope → verify → write handoff → stop.

## Compact-avoidance rule

Finish one small iteration and stop. Supervisor respawns for the next task.
Do not keep the goal open. Do not implement outside your writable scope.
