# Tigrinya Sample Corpus Notes

Locale: Tigrinya for Tigrinya-speaking Sweden newcomers.

## Register target

Formal public-information Tigrinya; prioritize official Swedish newcomer texts where available and keep UI copy shorter than long civic paragraphs.

## Translation risks

- Source coverage is thinner than some languages; do a fresh official-source pass before translating questions.
- Do not rely on English or Swedish word order.
- Require native review before shipping broad Tigrinya app copy, especially legal/civic terms.

## Reusable localization notes

- See `style-guide.md` for Tigrinya civic voice, UI patterns, glossary, humor guidance, and word-by-word audit checklist.
- Error encouragement should reassure and teach; never shame the learner.
- Source labels must say the app is based on/reference material, not an official exam authority.
- Institution names should teach the Swedish term when the term itself matters.
