# Tigrinya Civic Localization Style Guide

Locale: Tigrinya (`ti`) for Tigrinya-speaking Sweden newcomers.

## Purpose

Use this guide before translating Tigrinya UI, quiz questions, answer feedback,
or civic explanations. The goal is formal but clear public-information Tigrinya,
not English or Swedish word order written in Ge'ez script.

## Source-backed voice

Informationsverige Tigrinya pages provide a strong civic/newcomer register:

- section framing: `ብዛዕባ ሽወደን - ሓደ ንልልይ ምስ ሕብረተሰብ ዝኸውን ጽሑፍ።`
- topic openings: `እዚ ጽሑፍ ብዛዕባ ... እዩ።`
- learner preview: `... ከተንብብ ክትክእል ኢኻ።`
- rights language: `መሰል ኣለካ`, `ኵሉ ሰብ ... መሰል ኣለዎ`
- duties language: `ግዴታታት`, `ግቡኣት`
- example marker: `ንኣብነት`
- Sweden-specific framing: `ኣብ ሽወደን`
- reflective prompts and learning tone rather than jokes.

App copy should keep this public-information voice, but be shorter and more
scannable than long source paragraphs.

## Tone rules

1. Use Tigrinya consistently; do not rely on English/Swedish word order.
2. Keep civic/legal content formal, neutral, and precise.
3. Use `ኣብ ሽወደን` when a rule is Sweden-specific.
4. Teach Swedish institution names when the learner must recognize them.
5. For rights, duties, discrimination, gender, religion, and family, use calm
   public-service wording.
6. Feedback should reassure and teach; never shame the learner.
7. Never imply the app is an official exam authority or guarantees citizenship.

These are draft patterns and require native review before broad release.

| `እዚ ሕቶ ብዛዕባ እዩ...` | `እዚ ሕቶ ብዛዕባ ... እዩ።` | Natural topic sentence order. |
| `ናይ ልምምድ ክፍለ ግዜ ጀምር` | `ልምምድ ጀምር` | Short UI pattern. |
| `ትኽክል ኢኻ` | `መልስኻ ትኽክል እዩ` | Natural answer feedback. |
| `ጌጋ ኢኻ` | `ኣብዚ ግዜ መልስኻ ትኽክል ኣይነበረን` | Non-shaming. |
| `ምንጪ ንብረት` | `ምንጪ` / `መወከሲ ጽሑፍ` | Natural source wording. |

## Question and explanation style

- Start with the civic idea: `እዚ ሕቶ ብዛዕባ ... እዩ።`
- Use one or two short sentences in feedback.
- Use `ኣብ ሽወደን` for country-specific rules.
- Keep Swedish terms in parentheses on first mention when needed.
- Avoid translating every Swedish compound literally.

Example draft requiring native review:

```text
መልስኻ ትኽክል እዩ። ኣብ ሽወደን፡ መረጽቲ ኣብ ናጻን ፍትሓውን ምርጫ ሰልፍታትን ፖለቲከኛታትን ይመርጹ።
እዚ ክፋል ናይ ደሞክራሲ እዩ።
```

## Humor and encouragement

Use light encouragement, not jokes that may sound disrespectful in legal/civic
contexts.

Use draft patterns for native review:

- `ጸገም የለን፤ እዚ ሓሳብ ብልምምድ ይጸንዕ።`
- `ቀሪብካ ነይርካ። ፍልልይ መሰልን ግዴታን ኣስተውዕል።`
- `ጽቡቕ ስጉምቲ። እቲ ቀንዲ ሓሳብ እንደገና ንርአ።`

Avoid:

- sarcasm after mistakes;
- jokes about citizenship, passports, migration status, religion, gender, or
  family;
- direct translation of Swedish/English jokes;
- promises that practice guarantees an official result.

## Word-by-word audit checklist

Before accepting Tigrinya text, ask:

1. Does it sound like Tigrinya, not English/Swedish word order?
2. Are Ge'ez script, punctuation, and spacing readable in the app UI?
3. Are Swedish institutions preserved and explained, not replaced by another
   country's institutions?
4. Are `መሰላት`, `ግዴታታት`, `ግቡኣት`, `ዜግነት`, and `ኮሙን` used consistently?
5. Is learner feedback respectful and non-shaming?
6. Does any legal/civic statement need `ኣብ ሽወደን`?
7. Does the copy avoid promising exam success, citizenship, or passport
   outcomes?
8. Has a native Tigrinya reviewer checked legal/civic terms before release?
