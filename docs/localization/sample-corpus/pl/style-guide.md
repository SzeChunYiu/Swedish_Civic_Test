# Polish Civic Localization Style Guide

Locale: Polish for Polish-speaking learners/residents using a Sweden civic-test
app.

## Purpose

Use this guide before translating Polish UI, quiz questions, answer feedback, or
civic explanations. The goal is plain Polish public-service language that feels
written by a human, not a literal transfer from Swedish or English.

## Source-backed voice

Polish public-service pages and Polish Sweden Abroad pages share useful
patterns:

- direct address: `możesz`, `musisz`, `dostaniesz`, `sprawdź`, `pamiętaj`
- section labels: `Kto może...`, `Co musisz przygotować`, `Co musisz zrobić`,
  `Ile zapłacisz`, `Ile będziesz czekać`
- plain-language principles: start with what matters to the reader, use short
  sentences, avoid passive constructions, explain abbreviations at first use
- civic vocabulary: `prawa człowieka`, `demokracja`, `praworządność`,
  `państwo prawa`, `społeczeństwo obywatelskie`, `niedyskryminacja`
- Swedish-Polish public prose keeps Swedish institutions visible instead of
  replacing them with Polish equivalents.

App copy should be shorter than official pages, but it should keep their direct,
clear, user-oriented tone.

## Tone rules

1. Use standard Polish with clear, short sentences.
2. Avoid long legal sentence chains unless quoting or summarizing law.
3. Address the learner politely but directly; `ty` forms are acceptable in app
   guidance if the product tone uses them consistently.
4. Be precise about rights, duties, eligibility, and Swedish institutions.
5. Do not map Swedish institutions onto Polish ones.
6. Encourage without shaming; the learner is practicing, not being judged.
7. Do not imply the app is an official exam authority or a shortcut to Swedish
   citizenship.

## Mechanical translation repairs

| Mechanical | Better Polish | Why |
|---|---|---|
| `zrobić decyzję` | `podjąć decyzję` | Natural collocation. |
| `pytanie jest o...` | `to pytanie dotyczy ...` / `w tym pytaniu chodzi o ...` | Natural question framing. |
| `kliknij kontynuować` | `kliknij „Kontynuuj”` / `kontynuuj` | UI Polish needs command form or quoted label. |
| `to jest bardzo poprawne` | `to prawidłowa odpowiedź` / `dobrze` | Avoid English-style praise. |
| `nie zdałeś` | `tym razem odpowiedź nie była poprawna` | Non-shaming feedback. |
| `prawo mówi, że` | `zgodnie z prawem ...` / `ustawa przewiduje ...` | Legal/public-service register. |
| `w dniu dzisiejszym` | `dzisiaj` | Plain-language principle. |
| `celem jest aby` | `celem jest ...` / `chodzi o to, aby ...` | Avoid heavy bureaucratic phrasing. |
| `dokonać wyboru` | `wybrać` | Prefer simple verbs in UI. |
| `dedykowany dla` | `przeznaczony dla` | Gov.pl plain-language guidance. |

| Concept | Polish target | Notes |
|---|---|---|
| Sweden | `Szwecja` | Use `w Szwecji`. |
| civic orientation | `orientacja społeczna` / `wiedza o społeczeństwie` | Choose by context; explain if formal. |
| democracy | `demokracja` | Use `demokracja przedstawicielska` when relevant. |
| rule of law | `praworządność` / `państwo prawa` | Both appear in official Polish public prose. |
| Riksdag | `Riksdag (szwedzki parlament)` | First mention should teach the Swedish term. |
| Swedish Parliament | `szwedzki parlament` | Avoid `Sejm`, which is Polish-specific. |
| government | `rząd` | Do not confuse with `państwo`. |
| authority/agency | `urząd` / `organ państwowy` | Use exact agency name where known. |
| municipality | `gmina` / `samorząd gminny` | Explain Swedish `kommun` if needed: `gmina (kommun)`. |
| region | `region` | Swedish regional government level. |
| citizen | `obywatel` / `obywatelka` | Use plural forms when possible. |
| citizenship | `obywatelstwo szwedzkie` | For legal citizenship context. |
| rights | `prawa` | Pair with `obowiązki` where relevant. |
| duties/obligations | `obowiązki` / `zobowiązania` | `zobowiązania` is more formal/legal. |
| discrimination | `dyskryminacja` | For protected-ground legal/civic content. |
| non-discrimination | `niedyskryminacja` | Sweden Abroad EU/civic register. |
| gender equality | `równość płci` | Use for jämställdhet. |
| freedom of expression | `wolność wypowiedzi` / `wolność słowa` | Choose by source context. |
| source criticism | `krytyczna ocena źródeł` | For reliability-checking. |

## Question and explanation style

- Start with the key idea, then add the civic detail.
- Use `w Szwecji` when a rule is Sweden-specific.
- Explain Swedish terms on first mention instead of substituting Polish
  institutions.
- Prefer verbs over noun-heavy phrases: `wybierz`, not `dokonaj wyboru`.
- Use Polish quotation marks `„...”` for UI labels and quoted terms.

Example:

```text
Dobrze. W Szwecji wyborcy wybierają partie i polityków w wolnych wyborach.
To jedna z zasad demokracji przedstawicielskiej.
```

## Humor and encouragement

Polish civic copy can be warm and slightly conversational, but keep official and
sensitive topics restrained. Do not joke about migration status, citizenship,
passports, government offices, religion, gender, or minority rights.

Use:

- `Nic się nie stało — to pojęcie wymaga trochę praktyki.`
- `Było blisko. Zwróć uwagę na różnicę między prawem a obowiązkiem.`
- `Dobry krok. Teraz krótko powtórzmy najważniejszą myśl.`

Avoid:

- sarcasm about bureaucracy or waiting times
- English jokes translated literally
- over-formal jokes that sound like office paperwork
- promises that practice guarantees citizenship or passing an official test

## Word-by-word audit checklist

Before accepting Polish text, ask:

1. Is the sentence clear after one reading?
2. Can a noun-heavy phrase become a simple verb?
3. Are Swedish institutions preserved and explained accurately?
4. Are `prawa`, `obowiązki`, `zobowiązania`, and `odpowiedzialność` used
   consistently?
5. Is learner feedback respectful and non-shaming?
6. Does any legal/civic claim need `w Szwecji` or `zgodnie z prawem`?
7. Does the text avoid promising official exam or citizenship outcomes?
