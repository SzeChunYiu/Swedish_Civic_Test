# Polish Sample Corpus Notes

Locale: Polish for Polish-speaking Sweden residents.

## Register target

Polish plain public-service/legal-information prose for style, with Swedish official sources for institution facts.

## Seed sources

| Quality | Source | URL | Why use |
|---|---|---|---|
| official-register | Gov.pl public-service prose | https://www.gov.pl/web/witd-opole/dzialalnosc | Polish official register examples for tasks, rights, duties, procedures. |
| official-fact+register | Government.se How Sweden is governed | https://www.regeringen.se/other-languages/english---how-sweden-is-governed/ | Swedish institution fact anchor until Polish Swedish-source coverage is found. |

## Translation risks

- Do not map Riksdag or Swedish agencies to Polish institutions.
- Use Polish register for clarity, not long legal sentence chains.

## Reusable localization notes

- See `style-guide.md` for Polish civic voice, plain-language repairs, UI patterns, glossary, and word-by-word audit checklist.
- Error encouragement should reassure and teach; never shame the learner.
- Source labels must say the app is based on/reference material, not an official exam authority.
- Institution names should teach the Swedish term when the term itself matters.
