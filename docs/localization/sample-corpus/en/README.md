# English Sample Corpus Notes

Locale: Plain English for adult learners preparing for a Swedish civic/citizenship test.

## Register target

Clear learner-facing public information; avoid legalese unless the source requires it.

## Translation risks

- English is not a literal bridge for other languages; keep it natural and plain.
- Do not overclaim official status of this app.

## Reusable localization notes

- Error encouragement should reassure and teach; never shame the learner.
- Source labels must say the app is based on/reference material, not an official exam authority.
- Institution names should teach the Swedish term when the term itself matters.
- See `style-guide.md` for source-locale wording, glossary, UI patterns, humor boundaries, and the word-by-word audit gate.
