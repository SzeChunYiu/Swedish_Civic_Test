# 01 — Project Plan

## Project name

Working name: **Sweden Citizenship Test Prep**

Alternative names:

- Civic Sweden
- Sweden Civic Test Prep
- Medborgarskapsprovet Prep
- Sverige Civic Quiz
- Swedish Citizenship Civic Test

Recommended App Store name:

> Sweden Citizenship Test Prep

Subtitle:

> Swedish civic exam practice

## Product vision

Build a bilingual Swedish/English mobile learning app that helps people prepare for the Swedish citizenship civic knowledge test through:

- chapter-based study,
- practice questions,
- realistic mock exams,
- mistake review,
- Swedish audio,
- progress tracking,
- and UHR-referenced answer explanations.

The app should be low-cost to operate and should not depend on AI features.

## Problem

People preparing for the Swedish citizenship civic test will need:

- a clear way to study UHR's Swedish material,
- English support to understand concepts,
- repeated practice,
- confidence before the exam,
- mobile-friendly lessons,
- Swedish wording practice,
- and source-backed explanations.

PDF reading alone is not enough for many learners.

## Solution

Create a mobile app that converts the official study material into a structured, interactive learning journey.

The app will provide:

1. Swedish-first civic test practice.
2. English translations and explanations.
3. UHR references for trust.
4. Gamified practice to support retention.
5. Audio support for Swedish pronunciation.
6. Mock exam mode.
7. Low-cost monetization through ads and optional premium.

## Product principles

### 1. Trust first

The user must know the app is independent but based on public UHR study material.

### 2. Swedish is the exam language

The app can support English, but users must practise Swedish wording.

### 3. No fake authority

Never claim:
- official app,
- UHR-approved,
- real exam questions,
- guaranteed pass.

### 4. Explanations matter

Each answer should explain:

- why the correct option is right,
- why common wrong options are wrong,
- where the concept appears in UHR's material.

### 5. Low-cost operation

Avoid server-heavy features. Use local content and rule-based learning logic.

## Target users

| Segment | Needs |
|---|---|
| Citizenship applicants | Prepare for civic knowledge exam |
| New Swedish residents | Understand Swedish society |
| Users with weak Swedish | English support and Swedish audio |
| Busy workers/parents | Short daily practice |
| Anxious test-takers | Mock exams and readiness score |

## Core MVP

MVP should include:

- Android app
- iOS app
- Swedish/English toggle
- chapter list
- practice quiz
- mock exam
- answer explanations
- UHR reference cards
- mistake review
- progress tracking
- streaks
- XP
- Swedish audio button using device TTS
- AdMob integration
- privacy policy and terms
- clear independent-app disclaimer

## Out of scope for MVP

Do not include:

- AI tutor
- AI-generated live explanations
- chat interface
- community forum
- user-generated content
- full human-narrated lesson audio
- complex backend
- web dashboard unless needed for content review
- live sync across devices

## Key risk

The biggest risk is not technology. The biggest risk is **content quality**.

Every question must be accurate, neutral, understandable, and traceable to source material.
