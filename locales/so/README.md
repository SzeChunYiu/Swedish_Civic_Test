# Somali Locale Workspace

Locale: Somali (`so`) for Somali-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Somali app UI or question content is enabled yet; `so` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Somali app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Somali UI phrases for common study actions and feedback.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/so/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Somali copy should follow clear Informationsverige-style public information: respectful, explanatory, and natural Somali, not English or Swedish word order.
