# Arabic UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Keep `ar` disabled until all release blockers in `audit.md` and `docs/localization/readiness.json` are resolved.

## Study actions

| UI function | Arabic candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | ابدأ التدريب | ابدأ جلسة ممارسة | Short and natural. |
| Continue practice | تابع التدريب | استمر في الجلسة | Natural CTA. |
| Try again | حاول مرة أخرى | أعد المحاولة لذلك | Standard UI phrase. |
| Next question | السؤال التالي | المشكلة التالية | Use سؤال. |
| Previous question | السؤال السابق | سؤال قبل | Natural UI phrase. |
| Submit answer | أرسل الإجابة | قدّم جوابًا | Natural in app context. |
| Review explanation | اقرأ الشرح | راجع لوحة التفسير | Avoid mechanical UI noun phrase. |
| Review mistakes | راجع الأسئلة التي أخطأت فيها | راجع الأخطاء | Clear learner phrase. |
| Start mock exam | ابدأ الاختبار التجريبي | ابدأ الامتحان المزيف | Do not imply official test. |

## Feedback

| UI function | Arabic candidate | Avoid | Notes |
|---|---|---|---|
| Correct | صحيح. | أنت صحيح. | Short feedback. |
| Incorrect | ليست الإجابة الصحيحة تمامًا. | خطأ! | Gentle and less shaming. |
| Correct answer label | الإجابة الصحيحة | الخيار الصحيح من الإجابة | Natural quiz label. |
| Your answer label | إجابتك | جوابك الخاص | Natural. |
| Explanation | الشرح | لوحة التفسير | Short label. |
| Encouragement | لا بأس، سيساعدك الشرح على تذكّر الفكرة. | لقد فشلت. | Supportive tone. |
| Progress encouragement | لقد تقدمت خطوة أخرى اليوم. | تم تحسين تقدمك. | Natural warmth. |

## Progress and settings

| UI function | Arabic candidate | Avoid | Notes |
|---|---|---|---|
| Question progress | السؤال {current} من {total} | سؤال {current}/{total} only | Full phrase can be clearer with RTL. |
| Completed | أُنجز {done} من {total} | مكتمل {done} داخل {total} | Natural progress label. |
| Daily goal | الهدف اليومي | هدف يوم | Standard UI phrase. |
| Study streak | أيام الدراسة المتتالية | سلسلة الدراسة | Explain concept rather than literal streak. |
| Settings | الإعدادات | الضبطيات | Standard app label. |
| Question language | لغة الأسئلة | لغة السؤال | Plural for app setting. |
| Coming soon | هذا الإصدار قيد الإعداد | قريبًا جدًا | Public-information tone. |
| Local-only progress | يتم حفظ تقدمك في الدراسة على هذا الجهاز فقط. | تقدمك مخزّن محليًا | Plain privacy wording. |

## Source and authority labels

| UI function | Arabic candidate | Notes |
|---|---|---|
| Source material | مادة مرجعية | Needs native review. |
| Source citation | المصدر | Short citation label. |
| Based on UHR material | تم إعداد ذلك بالاستناد إلى مادة UHR المرجعية. | Do not imply official endorsement. |
