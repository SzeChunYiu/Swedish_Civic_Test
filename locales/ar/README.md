# Arabic Locale Workspace

Locale: Modern Standard Arabic (`ar`) for Arabic-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Arabic app UI or question content is enabled yet; `ar` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Arabic app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Arabic UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/ar/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Arabic copy should be clear Modern Standard Arabic in a public-information register: respectful, direct, and natural for adult learners, not colloquial Arabic and not English/Swedish word order.
