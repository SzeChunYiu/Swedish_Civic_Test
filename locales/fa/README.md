# Persian/Farsi Locale Workspace

Locale: Persian/Farsi (`fa`) for Persian-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Persian app UI or question content is enabled yet; `fa` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Persian app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Persian UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/fa/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Persian copy should use clear public-information Persian consistent with Informationsverige-style newcomer material. Decide explicitly whether a future release targets Iran Persian, Dari, or a neutral register before final UI/content review.
