# Persian/Farsi UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Keep `fa` disabled until all release blockers in `audit.md` and `docs/localization/readiness.json` are resolved.

## Study actions

| UI function | Persian candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | تمرین را شروع کنید | یک جلسه تمرین را شروع کنید | Natural concise CTA. |
| Continue practice | تمرین را ادامه دهید | ادامه دادن جلسه | Natural CTA. |
| Try again | دوباره تلاش کنید | آن را دوباره امتحان کنید | Standard UI phrase. |
| Next question | سؤال بعدی | مشکل بعدی | Use سؤال. |
| Previous question | سؤال قبلی | سؤال پیشین در UI | Natural app wording. |
| Submit answer | پاسخ را ثبت کنید | ارسال کردن یک جواب | Natural app action. |
| Review explanation | توضیح را بخوانید | پنل توضیح را بررسی کنید | Avoid mechanical UI term. |
| Review mistakes | سؤال‌هایی را که اشتباه پاسخ داده‌اید مرور کنید | اشتباهات را بازبینی کنید | Clear learner phrase. |
| Start mock exam | آزمون آزمایشی را شروع کنید | امتحان جعلی را شروع کنید | Do not imply official test. |

## Feedback

| UI function | Persian candidate | Avoid | Notes |
|---|---|---|---|
| Correct | درست است. | شما صحیح هستید. | Short feedback. |
| Incorrect | کاملاً درست نیست. | غلط! | Gentle and less shaming. |
| Correct answer label | پاسخ درست | گزینه صحیح جواب | Natural quiz label. |
| Your answer label | پاسخ شما | جواب خودتان | Formal learner copy. |
| Explanation | توضیح پاسخ | پنل توضیحات | Study context. |
| Encouragement | اشکالی ندارد؛ توضیح به شما کمک می‌کند نکته را به خاطر بسپارید. | شکست خوردید. | Supportive tone. |
| Progress encouragement | امروز یک قدم دیگر جلو رفتید. | پیشرفت شما بهینه شد. | Natural warmth. |

## Progress and settings

| UI function | Persian candidate | Avoid | Notes |
|---|---|---|---|
| Question progress | سؤال {current} از {total} | سؤال {current}/{total} only | Full phrase can be clearer with RTL. |
| Completed | {done} از {total} کامل شده است | کامل شده داخل | Natural progress label. |
| Daily goal | هدف روزانه | هدف روز | Standard. |
| Study streak | روزهای پیاپی مطالعه | استریک مطالعه | Explain concept. |
| Settings | تنظیمات | ستینگ‌ها | Standard app label. |
| Question language | زبان سؤال‌ها | زبان سوال | Plural for setting. |
| Coming soon | این نسخه در حال آماده‌سازی است | به زودی می‌آید | Public-information tone. |
| Local-only progress | پیشرفت آموزشی شما فقط در این دستگاه ذخیره می‌شود. | پیشرفت محلی ذخیره شد | Plain privacy wording. |

## Source and authority labels

| UI function | Persian candidate | Notes |
|---|---|---|
| Source material | مطالب مرجع | Needs native review. |
| Source citation | منبع | Short citation label. |
| Based on UHR material | این مطلب بر اساس مطالب مرجع UHR تهیه شده است. | Do not imply official endorsement. |
| Not official exam | این برنامه فقط برای تمرین است و آزمون رسمی نیست. | Must remain clear. |
| Citizenship rules warning | شرایط شهروندی/تابعیت ممکن است تغییر کند؛ آخرین اطلاعات را از اداره مهاجرت سوئد بررسی کنید. | Choose شهروندی/تابعیت after native/legal review. |
