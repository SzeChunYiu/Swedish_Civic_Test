# Polish Locale Workspace

Locale: Polish (`pl`) for Polish-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Polish app UI or question content is enabled yet; `pl` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Polish app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Polish UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/pl/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Polish copy should use plain official/public-service Polish. Use Polish government and ombudsman-style material for register, but Swedish/English official sources for Swedish civic facts. Do not import Polish institutions into Swedish civics.
