# Polish UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Keep `pl` disabled until all release blockers in `audit.md` and `docs/localization/readiness.json` are resolved.

## Study actions

| UI function | Polish candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | Rozpocznij ćwiczenie | Zacznij sesję praktyki | Natural CTA. |
| Continue practice | Kontynuuj ćwiczenie | Kontynuuj sesję | Natural CTA. |
| Try again | Spróbuj ponownie | Próbuj znowu | Standard UI phrase. |
| Next question | Następne pytanie | Kolejny problem | Use pytanie. |
| Previous question | Poprzednie pytanie | Pytanie wstecz | Natural UI phrase. |
| Submit answer | Zatwierdź odpowiedź | Wyślij odpowiedź | `Zatwierdź` often fits quiz UI. |
| Review explanation | Przeczytaj wyjaśnienie | Przejrzyj panel wyjaśnienia | Avoid mechanical UI term. |
| Review mistakes | Przejrzyj błędne odpowiedzi | Przejrzyj błędy | Clear learner phrase. |
| Start mock exam | Rozpocznij test próbny | Rozpocznij fałszywy egzamin | Do not imply official test. |

## Feedback

| UI function | Polish candidate | Avoid | Notes |
|---|---|---|---|
| Correct | Dobrze. | Jesteś poprawny. | Short feedback. |
| Incorrect | Nie do końca. | Źle! | Gentler tone. |
| Correct answer label | Prawidłowa odpowiedź | Poprawna opcja | Natural quiz label. |
| Your answer label | Twoja odpowiedź | Odpowiedź ciebie | Natural. |
| Explanation | Wyjaśnienie | Panel wyjaśnienia | Short label. |
| Encouragement | Nic się nie stało; wyjaśnienie pomoże zapamiętać tę zasadę. | Nie udało ci się. | Supportive tone. |
| Progress encouragement | Dziś zrobiłeś/zrobiłaś kolejny krok naprzód. | Twój progres został zoptymalizowany. | Needs gender-neutral review. |

## Progress and settings

| UI function | Polish candidate | Avoid | Notes |
|---|---|---|---|
| Question progress | Pytanie {current} z {total} | Pytanie {current}/{total} only | Full phrase is clear. |
| Completed | Ukończono {done} z {total} | Skompletowano | Natural progress label. |
| Daily goal | Cel dzienny | Dzienny cel maybe also ok | Standard. |
| Study streak | Dni nauki z rzędu | Streak nauki | Explain concept. |
| Settings | Ustawienia | Settings | Standard app label. |
| Question language | Język pytań | Język pytania | Plural for setting. |
| Coming soon | Ta wersja językowa jest w przygotowaniu | Już wkrótce | More public-information tone. |
| Local-only progress | Postępy w nauce są zapisywane tylko na tym urządzeniu. | Progres lokalny | Plain privacy wording. |
