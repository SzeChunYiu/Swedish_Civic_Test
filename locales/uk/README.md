# Ukrainian Locale Workspace

Locale: Ukrainian (`uk`) for Ukrainian-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. Ukrainian is currently corpus-only and is not in the runtime language picker; no Ukrainian app UI or question content is enabled.

Use these files before writing Ukrainian app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Ukrainian UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/uk/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Ukrainian copy should use standard Ukrainian public-service language. Avoid Russian calques, preserve Swedish institutions, and keep war/migration/citizenship topics restrained and factual.
