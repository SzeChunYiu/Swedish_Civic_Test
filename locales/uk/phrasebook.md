# Ukrainian UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Ukrainian is corpus-only and not currently in the runtime picker.

## Study actions

| UI function | Ukrainian candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | Почати тренування | Запустити практичну сесію | Short CTA. |
| Continue practice | Продовжити тренування | Продовжити сесію практики | Short CTA. |
| Try again | Спробувати ще раз | Пробувати знову | Natural UI phrase. |
| Next question | Наступне питання | Слідуюче питання | Avoid calque. |
| Previous question | Попереднє питання | Питання назад | Natural UI phrase. |
| Submit answer | Підтвердити відповідь | Відправити відповідь | Quiz confirmation. |
| Review explanation | Прочитати пояснення | Переглянути панель пояснення | Plain learner action. |
| Review mistakes | Переглянути помилки | Переглянути фейли | Respectful tone. |
| Start mock exam | Почати пробний іспит | Почати офіційний іспит | Must not imply official test. |

## Feedback

| UI function | Ukrainian candidate | Avoid | Notes |
|---|---|---|---|
| Correct | Правильно. | Ти правильний. | Short feedback. |
| Incorrect | Цього разу відповідь неправильна. | Ви провалили. | Gentle, non-shaming. |
| Correct answer label | Правильна відповідь | Коректна опція | Natural quiz label. |
| Your answer label | Ваша відповідь | Відповідь вас | Natural. |
| Explanation | Пояснення | Панель пояснення | Short label. |
| Encouragement | Нічого страшного — це поняття потребує трохи практики. | Ви не впоралися. | Supportive tone. |
| Progress encouragement | Добрий крок. Тепер коротко повторимо головну ідею. | Гарантовано складете іспит | No guarantee language. |

|---|---|---|---|
| Question progress | Питання {current} з {total} | {current}/{total} only | Full phrase is clearer. |
| Completed | Виконано {done} з {total} | Скомплектовано | Natural progress label. |
| Daily goal | Щоденна ціль | Денна мета maybe context-specific | App setting. |
| Study streak | Дні навчання поспіль | streak | Explain concept. |
