# Tigrinya Locale Workspace

Locale: Tigrinya (`ti`) for Tigrinya-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Tigrinya app UI or question content is enabled yet; `ti` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Tigrinya app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Tigrinya UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/ti/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Tigrinya copy should sound like formal but clear public-information Tigrinya, not Swedish or English word order in Ge'ez script. Use Informationsverige Tigrinya pages for Swedish civic register when possible, keep Swedish institution names visible when learners must recognize them, and require native review before shipping legal/civic terms.
