# Turkish Glossary

Status: phase-1 glossary. Use with `docs/localization/sample-corpus/tr/style-guide.md`.

## Source notes

Sweden Abroad Turkish pages provide Swedish-authority Turkish register for migration and public-service process language. Useful patterns include:

- `İsveç` for Sweden.
- `İsveç Göçmen Dairesi` for the Swedish Migration Agency in service pages.
- `oturum izni` for residence permit.
- `başvuru yapmak`, `gerekli belgeler`, `başvuru ücreti`, and `karar verilir` for application/process copy.
- `İnternet üzerinden başvuru` for online application context.

For democracy, rights, citizenship, and governance facts, use Swedish/English official fact sources and translate into Turkish with this register. Do not import Turkey-specific institutions into Swedish civics.

## False friends and style warnings

- Do not write `karar yapmak`; use `karar vermek` or passive `karar verilir`.
- Do not write `vatandaşlık için uygulamak`; use `İsveç vatandaşlığına başvurmak`.
- Do not use casual `sen`, `hadi`, or internet slang in core learning copy.
- Do not translate Swedish institutions into Turkey-specific equivalents.
- Re-check time-sensitive citizenship facts before release.
