# Turkish Locale Workspace

Locale: Turkish (`tr`) for Turkish-speaking adults preparing for Swedish civic/citizenship study material.

Status: phase-1 glossary/phrasebook workspace. No Turkish app UI or question content is enabled yet; `tr` must remain `available: false` until the readiness ledger is complete.

Use these files before writing Turkish app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Turkish UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/tr/style-guide.md` — register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Turkish copy should use formal, clear public-information Turkish: respectful `siz` register when addressing the reader, natural Turkish verb patterns, and Sweden-specific explanations rather than Turkey-specific institutional substitutions.
