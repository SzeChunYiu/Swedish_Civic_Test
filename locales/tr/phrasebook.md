# Turkish UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Keep `tr` disabled until all release blockers in `audit.md` and `docs/localization/readiness.json` are resolved.

## Study actions

| UI function | Turkish candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | Alıştırmaya başlayın | Bir pratik oturumu başlat | Natural formal CTA. |
| Continue practice | Alıştırmaya devam edin | Session'a devam et | Avoid English loan. |
| Try again | Tekrar deneyin | Onu yeniden dene | Standard UI phrase. |
| Next question | Sonraki soru | Bir sonraki problem | Use soru. |
| Previous question | Önceki soru | Geri soru | Natural UI phrase. |
| Submit answer | Cevabı gönderin | Bir cevap submit edin | Keep Turkish. |
| Review explanation | Açıklamayı okuyun | Açıklama panelini incele | Avoid mechanical UI term. |
| Review mistakes | Yanlış yaptığınız soruları gözden geçirin | Hataları review edin | Clear learner phrase. |
| Start mock exam | Deneme sınavını başlatın | Sahte sınavı başlat | Do not imply official test. |

## Feedback

| UI function | Turkish candidate | Avoid | Notes |
|---|---|---|---|
| Correct | Doğru. | Sen doğrusun. | Short feedback. |
| Incorrect | Tam doğru değil. | Yanlış! | Gentler tone. |
| Correct answer label | Doğru cevap | Doğru opsiyon | Natural quiz label. |
| Your answer label | Cevabınız | Senin cevabın | Formal register. |
| Explanation | Açıklama | Açıklama paneli | Short label. |
| Encouragement | Sorun değil; açıklama konuyu hatırlamanıza yardımcı olur. | Başarısız oldunuz. | Supportive tone. |
| Progress encouragement | Bugün bir adım daha ilerlediniz. | Progress'iniz optimize edildi. | Natural warmth. |
