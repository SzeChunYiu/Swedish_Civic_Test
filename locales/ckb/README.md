# Central Kurdish / Sorani Locale Workspace

Locale: Central Kurdish / Sorani (`ckb`) in Arabic-based Kurdish script.

Status: phase-1 glossary/phrasebook workspace. Sorani is currently corpus-only and is not in the runtime language picker; no Sorani app UI or question content is enabled.

Use these files before writing Sorani app copy:

- `glossary.md` — sourced civic and UI term decisions.
- `phrasebook.md` — candidate Sorani UI phrases for common study actions, feedback, source labels, and safe encouragement.
- `audit.md` — checked decisions and remaining release blockers.
- `docs/localization/sample-corpus/ckb/style-guide.md` — source limitations, register, mechanical-translation repairs, culture/humor rules, and word-by-word audit checklist.

Because no full official Swedish civic-orientation corpus in Sorani has been identified yet, use Sorani Swedish public-service/education material for language style and Swedish/English official sources for Swedish civic facts. Require native Sorani review before any broad UI/content release.
