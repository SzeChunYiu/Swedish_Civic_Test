# Simplified Chinese UI Phrasebook

Status: phase-1 candidate copy, not wired into the app. Keep `zh-Hans` marked
`available: false` until full UI, question content, accessibility labels, and
native review are complete.

Use this phrasebook with `glossary.md` and
`docs/localization/sample-corpus/zh-Hans/style-guide.md`. It records native
Simplified Chinese candidates for high-frequency app UI so future work does not
fall back to mechanical English/S Swedish wording.

## Core study actions

| UI function | zh-Hans candidate | Avoid | Notes |
|---|---|---|---|
| Start practice | 开始练习 | 开始一个练习会话 | Short study-app wording. |
| Resume practice | 继续练习 | 恢复练习会话 | Natural learner CTA. |
| Continue learning chapter | 继续学习第 {chapter} 章 | 继续章节 | Include object for clarity. |
| Try again | 再试一次 | 重试它 | Friendly, not technical. |
| Next question | 下一题 | 下一个问题 | Study context uses 题. |
| Previous question | 上一题 | 之前的问题 | Study context uses 题. |
| Submit answer | 提交答案 | 提交一个回答 | Natural object. |
| Review explanation | 查看解析 | 检查解释 | Study-app term. |
| Review mistakes | 复习错题 | 审查错误 | Native learning term. |
| Start mock exam | 开始模拟考试 | 开始假的考试 | Mock exam is 模拟考试. |
| Finish exam | 交卷 | 完成考试 | Use in exam flow only; otherwise use 完成. |

## Feedback and encouragement

| UI function | zh-Hans candidate | Avoid | Notes |
|---|---|---|---|
| Correct | 答对了。 | 你是正确的。 | Short and warm. |
| Incorrect | 这次选错了。 | 你是不正确的。 | Less accusatory. |
| Correct answer label | 正确答案 | 正确的选项 | Common quiz label. |
| Your answer label | 你的答案 | 你的回答 | Use 你 in direct feedback; avoid over-formal 您 in practice feedback. |
| Explanation | 答案解析 | 解释面板 | Native study-app label. |
| Keep going | 继续加油 | 保持进行 | Warm but not childish. |
| Small progress | 今天又前进了一小步。 | 你已经优化了你的进度。 | Natural encouragement. |
| Mistake reassurance | 错一次没关系，把这个知识点记牢就好。 | 错误是可以接受的。 | Localized warmth. |

## Source and authority labels

| UI function | zh-Hans candidate | Avoid | Notes |
|---|---|---|---|
| Source material | 参考材料 | 源材料 | Natural label. |
| Source citation | 资料来源 | 来源引用 | Short label. |
| Based on UHR reference | 依据 UHR 参考材料整理 | 基于 UHR 源 | Do not imply official endorsement. |
| Not official exam | 本应用仅供练习，并非官方考试。 | 这是官方考试练习。 | Must be explicit. |
| Citizenship rules warning | 公民身份要求可能会变化，请以瑞典移民局最新信息为准。 | 这些规则总是正确。 | Time-sensitive legal facts. |
| External link | 打开外部网站 | 开放外部链接 | Natural mobile UI. |

## Civic question stems

| English function | zh-Hans pattern | Notes |
|---|---|---|
| What does X mean in Sweden? | 在瑞典，{term} 是什么意思？ | Use for definitions. |
| Which institution...? | 哪个机构负责{action}？ | Use 机构 for public bodies. |
| Why is X important? | 为什么{term}很重要？ | Good for concept checks. |
| Which statement is correct? | 以下哪项说法正确？ | Standard multiple-choice wording. |
| What can you do if...? | 如果{scenario}，你可以怎么做？ | Avoid legal advice beyond source. |
| According to source material... | 根据参考材料，{question} | Use sparingly; app should not sound like a citation dump. |

## Culture and humor candidates

Use only after native review.

| Function | Candidate | Notes |
|---|---|---|
| Gentle break | 先喝口水，再看下一题。 | Neutral, not culture-specific. |
| Repeated Swedish term | Riksdag 这个词会反复出现，记住它很划算。 | Helps with Swedish institutional vocabulary. |
| Light progress | 今天又前进了一小步。 | Safe encouragement. |

Avoid direct translations of fika, midsummer, or Swedish animal/food jokes unless
the Swedish culture term is part of the teaching point.
